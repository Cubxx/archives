# 游戏事件

进入游戏 **自动全屏**

## 资源

玩家靠近 boss / bullet 产生 **分数资源**  
游戏根据玩家事件 生成 **生命资源** / **炸弹资源**

## 玩家事件

**掉血：** 💣 数变为 2 🧡 数-1  
**开大：** 💣 数-1 开大瞬间吸收界面所有资源  
**吸收：** 玩家位于界面上方可 吸收界面所有资源

# 游戏关卡

共 6 关，根据关卡逻辑自动进入下一关
